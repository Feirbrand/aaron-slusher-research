## What It Is

Neural Access Method™ (NAM) is the transmission protocol layer within Neural Formation Architecture. It describes how to access an adaptive system at the point where change is possible and deliver correction through existing procedural, emotional, or identity-linked pathways.

The four-step protocol: ACCESS → REFRAME → SIMPLIFY → IGNITE.
