## Public Role

Elevation Grid is the **diagnostic map** within the Adapted Architecture research track. It maps state, access, and developmental position — identifying where in the formation arc a person or system currently sits.

In the public architecture stack:

```
Adapted Architecture    ← public outcome frame (the front door)
  └── Neuroformation   ← methodology process
  └── Elevation Grid   ← diagnostic map (THIS METHOD)
  └── Neural Access Method  ← access intervention protocol
```
